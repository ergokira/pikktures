
# Pikktures

This is a simple Gallery web project with gulp-sass for precompile Sass and browser reload with browsersync

## Quick Start

```bash
$ npm install
$ bower install

```

### Includes ###

> - Bootstrap-sass
> - Jquery


### Tasks ###

> - Sass Compilation (gulp-sass)
> - Browser reload (browsersync)
> - Start the server (gulp serve or gulp)
